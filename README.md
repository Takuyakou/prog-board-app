# README  

## Description(まとめ)  
Docker環境下でRailsで作成した簡易掲示板アプリです。  

## Features（機能一覧）  
* 一覧表示機能
* 掲示板の作成機能
* 掲示板の削除機能
* 掲示板の投稿機能
* ログイン、ログアウト認証
* 画像アップロード機能
* タグ付け機能
* コメント機能
* ユーザー登録機能

## Requirement(技術一覧)
### Docker
* Version 19.03.4
* API version 1.40
* Go version go1.12.10
* OS/Arch darwin/amd64
